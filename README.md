# Convex + Expo Starter Code

A ready-to-go starter template for building real-time apps with [Convex](https://convex.dev) and [Expo](https://expo.dev). Works on **iOS**, **Android**, and **Web** — all from the same codebase.

## What's Included

- A working **real-time chat app** as a starting point (swap it out for whatever you're building)
- **Convex backend** already wired up — database schema, queries, mutations, real-time subscriptions
- **Expo frontend** — runs on your phone via Expo Go or in your browser as a web app
- Everything pre-configured so you can go from clone to running app in under 3 minutes

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React Native (Expo) |
| Backend | Convex (real-time database + serverless functions) |
| Platforms | iOS, Android, Web |

## Get Started

```bash
git clone <your-repo-url> my-project
cd my-project
npm install
npx convex dev          # Terminal 1 — starts the backend
npx expo start          # Terminal 2 — starts the app
```

Scan the QR code with your phone, or press `w` to open in your browser.

**See [setup.md](setup.md) for the full guide** — including web setup, Vercel deployment, and App Store publishing.

## Project Structure

```
App.js                  ← Your app's UI
convex/
  schema.js             ← Database schema (tables and fields)
  messages.js           ← Backend functions (queries and mutations)
  _generated/           ← Auto-generated by Convex (don't edit)
app.json                ← Expo config (app name, icons, etc.)
setup.md                ← Detailed setup and deployment guide
```

## What Can I Build With This?

This starter gives you a real-time backend and a cross-platform frontend. Swap out the chat UI for whatever you want:

- Chat / messaging apps
- Collaborative tools (shared lists, boards, docs)
- Live dashboards
- Social apps
- Multiplayer games
- Any app that needs real-time data sync

The Convex backend gives you a database, real-time subscriptions, and serverless functions out of the box — no server to manage.
